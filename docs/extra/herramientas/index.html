Herramientas y tecnologias

En este apartado citare varias herramientas/tecnologias y las agrupare teniendo en cuenta su finalidad.

Frameworks

framework PHP: los framework de PHP nos reducen la cantidad de codigo a escribir gracias a que nos proporcionan bibliotecas de codigo que cuentan con funciones de uso comun.

Django es un framework de alto nivel escrito en Phyton, nos permite desarrollar sitios web de manera rapida.

Vue es un framework perteneciente a javascript,su funcion es la construccion de interfaces de usuario y aplicaciones que cuentan con una sola pagina(es de codigo abierto).

Angular es un framework empleado en la creacion y modificacion de aplicaciones web de una sola pagina(es de codigo abierto).

Hugo es un framework para creación de sitios web estaticos, esta totalmente escrito en Go. Este sitio a sido creado con la ayuda de Hugo.

Generadores de sitios web estaticos

vuepress es un generador de sitios web estaticos, al principo esta hecho para complementar Vue.js.

Pelican es otro generador de sitios web estaticos libre y escrito en python.

Jekyll al igual que los anteriores tambien permite crear sitios web estaticos de forma simple, se destaca por crear sitios con capacidades de blog.

Gatsby, o GatsbyJS, es un generador de sitios web estáticos, se apoya en React JS un biblioteca de Javascript,permite crear sitios web con varias funcionalidades de forma sencilla.

Lenguajes

JavaScriptes un lenguaje de programacion interpretado y orientado a objetos, se suele usar para añadir caracteristicas interactivas a sitios web.

Css es un lenguaje de diseño grafico que nos permite determinar el estilo de un documento HTML y derivados.

PHP es un lenguaje de programacion de codigo abierto muy util a la hora de desarrollar web’s.
