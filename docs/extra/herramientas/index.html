Herramientas y tecnologías

En este apartado citaré varias herramientas/tecnologías y las agruparé teniendo en cuenta su finalidad.

Frameworks

framework PHP: los framework de PHP nos reducen la cantidad de código a escribir gracias a que nos proporcionan bibliotecas de código que cuentan con funciones de uso común.

Django es un framework de alto nivel escrito en Phyton, nos permite desarrollar sitios web de manera rápida.

Angular es un framework empleado en la creación y modificación de aplicaciones web de una sola página(es de código abierto).

Generadores de sitios web estáticos

Hugo es un framework para creación de sitios web estáticos, está totalmente escrito en Go. Este sitio ha sido creado con la ayuda de Hugo.

vuepress es un generador de sitios web estáticos, al principio está hecho para complementar Vue.js.

Pelican es otro generador de sitios web estáticos libre y escrito en python.

Jekyll al igual que los anteriores también permite crear sitios web estáticos de forma simple, se destaca por crear sitios con capacidades de blog.

Gatsby, o GatsbyJS, es un generador de sitios web estáticos, se apoya en React JS una biblioteca de Javascript, permite crear sitios web con varias funcionalidades de forma sencilla.

Lenguajes

JavaScriptes un lenguaje de programación interpretado y orientado a objetos, se suele usar para añadir características interactivas a sitios web.

Css es un lenguaje de diseño gráfico que nos permite determinar el estilo de un documento HTML y derivados.

PHP es un lenguaje de programación de código abierto muy útil a la hora de desarrollar web’s.

Bibliotecas

Vue es un framework perteneciente a javascript, su función es la construcción de interfaces de usuario y aplicaciones que cuentan con una sola página(es de código abierto).

jQuery es una biblioteca multiplataforma de javascript, su función es simplificar la interacción con el HTML y la manipulación del árbol DOM.

Bootstrap es una biblioteca que se enfoca en JS,HTML y CSS y nos permite desarrollar páginas web de una forma más simple.

otras herramientas

Babel es una herramienta que nos ofrece la posibilidad de transformar nuestro código JS actual a un código Javascript para que cualquier navegador o versión de Node.js sea capaz de interpretarlo.

Polyfill a diferencia de Babel que transforma el código, Polyfill proporciona una funcionalidad moderna a los navegadores desactualizados que no la admiten de forma nativa.

Webpack es un conjunto de módulos principalmente hechos para javaScript, nos permiten empaquetar y exportar ficheros para la buena funcionalidad de un proyecto.

Prototype es un conjunto de varias herramientas que nos permiten diseñar teniendo a nuestra disposición datos reales.
